# Shadowverse-diy

Shadowverse-diy 是一个影之诗卡牌 DIY 工具，旨在提供一个更规范便捷的方式来自定义影之诗卡牌。

## 开发目的

1. **学习**：通过开发这个项目，接触和学习一下electron。

2. **解决问题**：现接触到的工具无法有效地编写大量的卡牌描述，我希望能解决这个痛点。

3. **便捷的输入**：提供一个更便捷的方式来输入和编辑卡牌的能力关键字。
